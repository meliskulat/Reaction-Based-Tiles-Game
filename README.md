# Reaction-Based-Tiles-Game
Developed an interactive browser-based reaction game using JavaScript and DOM manipulation. The game features a 4x4 dynamic grid where a random tile turns black until clicked within a 10-second time limit, challenging users to react quickly. The game values less time passed between clicking the black tiles more, as the time between correct clicks increase, the point earned from clicking the black box drops. Implemented real-time scoring logic where faster clicks yield higher points, along with a high score system that updates automatically when a new record is achieved.s


## Project Overview 

The key aspects of this project include:

- The game features an interactive 4x4 grid where tiles are activated randomly using JavaScript to test player actions
- A real-time scoring logic is implemented with a system that rewards faster reaction times, each tile offers maximum 10 points
-High score persistence is applied by using localStorage to save and display the best performance     
- The user experience is enhanced with smooth CSS transitions, confetti effect for new high score
- Automatic game management handles the 10-second timer and it states a prompt to play again for the user
